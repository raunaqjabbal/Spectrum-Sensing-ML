LaTeX Style file for the Indian Institute of Information Technology and Management, Gwalior 
 
This directory contains

* readme - this file
* table-figure - Latex code for figures and tables
* logo.pdf - official IIIT Gwalior logo
* refs.bib - file for bibliography
* chapter[1-5].tex - file for each chapter
* appendix[A-B].tex - file for each appendix
* mythesis.tex  - mail file putting every chapter together

Modify the chapter.tex files for writing the chapters. Compile every chapter.tex file before compiling the 
mythesis.tex file. Modify mythesis.tex for other informations. The main class file thesis.cls must not be
touched.
